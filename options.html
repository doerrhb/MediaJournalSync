📊 Google Sheets Sync

Connect to Google Sheets via a deployed Apps Script. Each site will append to its own tab (Movies, Books, etc.).

Setup Instructions (click to expand)
1. Open your Google Sheet and go to Extensions → Apps Script
2. Paste this code into the editor, replacing any existing code:
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); // Find the tab by name — NEVER creates a new tab if it doesn't exist. // This protects your existing data. var sheet = ss.getSheetByName(data.tab); if (!sheet) { return ContentService .createTextOutput(JSON.stringify({ success: false, error: 'Tab "' + data.tab + '" not found. Check your tab name in extension settings.' })) .setMimeType(ContentService.MimeType.JSON); } // Duplicate check: scan column A (title) and column B (date) for a match. // Won't add a row if title+date already exists anywhere in the sheet. if (data.row && data.row.length >= 2) { var existingData = sheet.getDataRange().getValues(); for (var i = 0; i < existingData.length; i++) { var rowTitle = (existingData[i][0] || "").toString().trim().toLowerCase(); var rowDate = (existingData[i][1] || "").toString().trim().toLowerCase(); var newTitle = (data.row[0] || "").toString().trim().toLowerCase(); var newDate = (data.row[1] || "").toString().trim().toLowerCase(); if (rowTitle === newTitle && rowDate === newDate) { return ContentService .createTextOutput(JSON.stringify({success: true, skipped: true, reason: "Duplicate entry"})) .setMimeType(ContentService.MimeType.JSON); } } } // Safe to append — all existing rows are untouched. sheet.appendRow(data.row); return ContentService .createTextOutput(JSON.stringify({success: true, skipped: false})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({success: false, error: err.message})) .setMimeType(ContentService.MimeType.JSON); } }
3. Click Deploy → New deployment
4. Set type to Web app, execute as Me, access to Anyone
5. Click Deploy, authorize, and copy the Web app URL
6. Paste that URL below and save.

Leave blank to disable Sheets sync. Only CSV downloads will occur.

Sheet Tab Names

⚠️ These must exactly match your existing Google Sheet tab names (case-sensitive) to avoid creating duplicate tabs and losing your data.

📁 Image Download Folders

Where images are saved inside your Firefox Downloads folder. These become the default path shown in the review panel — you can also edit the path per entry before confirming.

Paths are relative to your Firefox Downloads folder. e.g. Images/movies → Downloads/Images/movies/

🔗 Custom Diary URLs

Your personal diary page URLs. Used when clicking "Open All Source Pages."
